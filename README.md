# 🤖 AI Agent Assistant

An AI-powered assistant built with React + TypeScript + Google Gemini, featuring real function calling, conversation memory, and live weather data.

## ✨ Features

- **AI Agent with Function Calling** — 4 working tools
  - 🧮 Calculator (math expressions)
  - 🕐 Current time (IST)
  - 🎲 Random number generator
  - 🌤️ Real-time weather (OpenWeatherMap API)
- **Conversation Memory** — remembers context across messages
- **Response Caching** — saves API calls for repeated queries
- **Streaming Effect** — ChatGPT-style typing animation
- **Custom System Prompt** — personalized AI personality
- **Modern UI** — ChatGPT-inspired design

## 🛠️ Tech Stack

- React 18 + TypeScript
- Vite
- Google Gemini API (gemini-2.5-flash)
- OpenWeatherMap API

## 🚀 Setup

1. Clone the repo
2. `npm install`
3. Create `.env` file:
4. `npm run dev`
